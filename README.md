## 试验记录自动生成系统

### 使用方法：
所有数据文件保存在程序所在目录下
- 把照片分类放入目录“\未打印照明”、“\未打印标志”
- 打开程序，点击“读取”，自动将照片编号写入“任务单.csv”（上面是照明、下面是标志）
- 打开“任务单.csv”，在每个编号后面录入型号

> “任务单.xls”设置自动更正选项，去掉大写自动转小写
录入数据格式为：
（照明）编号	型号
（中间空一行）
（标志）编号	型号

- 打开程序，点击“生成”，显示生成完成后点击“退出”
- 查看生成的文档，打印
