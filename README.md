## 试验记录自动生成系统

### 使用方法：
所有数据文件保存在程序所在目录下
- 打开“任务单.xls”表
(存为xls格式是为了保存以下设置信息)
- 设置第一列格式为自定义："2014"@
（自动添加年份）
- 设置自动更正选项，去掉大写自动转小写
- 录入数据，格式为：
```
（照明）编号	型号
（中间空一行）
（标志）编号	型号
```
- 另存为csv格式（纯文本，逗号分隔，便于编程读取）
- 确保为"ANSI"格式编码（用notepad++打开可以查看）
- 打开FAGS程序，点击“生成”
- 查看生成的文件，无误后点击“退出”
